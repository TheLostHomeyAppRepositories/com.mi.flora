Cette appli intègre le `Xiaomi mi flora sensor` et le `Xiaomi mi flora ropot` dans Homey.

1. Installe l'appli
2. Ajoute le(s) appareil(s) à Homey
3. Configure les seuils dans la configuration de l'appareil
4. Crée un flux avec l'une des cartes.

Tu peux configurer le délai entre les sondages dans les paramètres de l'appli.
